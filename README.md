Description
This is a PHP and MySQL web application where users can register, log in, view games, and submit reviews. Admin users can manage games and users.

How to Install and Run
Install XAMPP
Download and install XAMPP for Windows.

Put the project in the correct folder
Move the entire Project3 folder into:
C:\xampp\htdocs\

Start required services
Open the XAMPP Control Panel and start:
Apache
MySQL

Import the database
Go to:
http://localhost/phpmyadmin


Click Import
Select: Project3/sql/project3_db.sql
Click Go

Confirm the database
You should see a database named project3_db with tables:
users
games
reviews

Launch the site
Open:
http://localhost/Project-3/setup.php
